# AutoDock Vina 1.2.0: Expanded Docking Capabilities

## Cite_Key
eberhardt2021autodock

## Problem
AutoDock Vina lacked support for macrocycles, explicit water molecules, multi-ligand docking, and scripting interfaces compared to other AutoDock Suite tools.

## Method
Implementation of new docking methods (macrocycle support, explicit waters), AutoDock4.2 scoring function integration, batch docking mode, and Python bindings in Vina 1.2.0.

## Data
Molecular docking benchmarks; open-source codebase on GitHub.

## Metrics
Docking accuracy, scoring function performance, computational throughput in batch mode.

## Findings
Vina 1.2.0 supports macrocyclic ligands, explicit water molecules, AutoDock4.2 scoring, simultaneous multi-ligand docking, batch mode, and Python bindings. Represents unification of AutoDock4 and Vina feature sets.

## Limitations
Performance comparisons with other docking programs not fully detailed in abstract; new features may require additional validation on diverse target classes.

## Citation
Eberhardt et al. (2021). AutoDock Vina 1.2.0. J Chem Inf Model. doi:10.1021/acs.jcim.1c00203
